# Command: translate

See `commands/record.md` § translate — documented together because `translate` also processes video scripts produced by `record`.

Quick reference:

**Purpose**: translate drafts + video scripts from source locale to each `locales.targets`.

**Invocation**: `translate` / `tr`. Required when targets non-empty; runs after `edit`, before `deploy`.

**Review**: `batch` default, `--per-block` safer.

**Detail**: `references/translate-reference.md` (block rules, review modes). Glossary: `templates/GLOSSARY_TEMPLATE.yaml`. FPT vocabulary: `fpt/FPT_TEMPLATES.md` § Part 5.
