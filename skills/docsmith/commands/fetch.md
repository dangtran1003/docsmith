# Command: fetch

See `commands/module.md` § fetch — fetch is documented alongside module since they share source-handling logic.

Quick reference:

**Purpose**: pull external knowledge sources (Notion / GitHub / GDrive / URL / file) into `documentation/.cache/sources/`, update `sources.lock.yaml`.

**Invocation**: `fetch` / `f`

**Flags**: (none)=all, `--module <n>`, `--source <id>`, `--force`.

**Detail**: `references/intake-reference.md` § External source fetching (per-type behavior, auth, env vars).
